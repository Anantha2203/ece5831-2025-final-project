Final project submission
# Emotion Detection in Text Using Pattern Recognition and Neural Networks

A compact, reproducible pipeline for text emotion detection (Happy, Sad, Angry, Neutral) with a Gradio chatbot UI.
Models: a TF-IDF + Logistic Regression baseline and a BiLSTM classifier.
Deliverables include the final paper, slides, recorded talk, demo video, dataset link, and an executed notebook.
